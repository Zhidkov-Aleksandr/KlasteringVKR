# Кластерный анализ цифровизации регионов РФ

Интеллектуальная система оценки уровня цифровизации субъектов РФ на основе методов машинного обучения (K-Means, PCA).

## 🚀 Запуск через Docker (Рекомендуемый способ)

Теперь приложение работает как красивый и современный веб-интерфейс, который запускается в браузере. Для этого вам не нужно устанавливать Python или настраивать окружение, достаточно только установленного **Docker**.

### Шаг 1: Установка Docker
Если у вас еще не установлен Docker, скачайте и установите **Docker Desktop** для вашей ОС с официального сайта: [https://www.docker.com/products/docker-desktop/](https://www.docker.com/products/docker-desktop/)

### Шаг 2: Запуск приложения

Откройте терминал (командную строку) в папке с проектом `KlasteringVKR` и выполните команду:

```bash
docker-compose up --build
```

### Шаг 3: Открытие интерфейса

После того как процесс сборки и запуска завершится, откройте ваш браузер и перейдите по адресу:

👉 **[http://localhost:8502](http://localhost:8502)**

## 💻 Использование системы

1. **Загрузка данных:** В левой панели (сайдбаре) нажмите "Browse files" и выберите ваш Excel-файл (например, `Данные по федеральным округам.xlsx`).
2. **Выбор параметров:** Выберите год для анализа в выпадающем списке.
3. **Запуск модулей:** В центральной части экрана выберите один из трех уровней анализа:
   - Макроуровень (Федеральные округа)
   - Мезоуровень (Внутриокружной)
   - Микроуровень (Все субъекты РФ)
4. **Анализ результатов:** После завершения расчетов, прокрутите страницу вниз — там появятся красивые графики, диаграммы, тепловые карты и интерактивные таблицы с возможностью скачивания!

## 🔧 Запуск без Docker (Классический способ)

Если вы хотите запустить веб-интерфейс локально через Python:

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите Streamlit-приложение:
```bash
streamlit run app.py
```
