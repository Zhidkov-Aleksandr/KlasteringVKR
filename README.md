# 🇷🇺 Платформа кластерного анализа цифровизации регионов РФ

Интеллектуальная информационно-аналитическая система для многомерного статистического анализа и сегментации субъектов Российской Федерации по уровню цифрового развития. Проект реализован в рамках выпускной квалификационной работы (ВКР).

## 🚀 Основные возможности
*   **Многоуровневый анализ:** Поддержка Макро- (ФО), Мезо- (внутри ФО) и Микроуровня (все субъекты РФ).
*   **Интеллектуальная кластеризация:** Использование алгоритма **K-Means** с автоматической логической сортировкой кластеров (Лидеры → Потенциал → Аутсайдеры).
*   **Оптимизация параметров:** Определение оптимального числа групп методом «локтя» с визуальным выделением рекомендуемого значения (k=3).
*   **Продвинутая визуализация:**
    *   **Интерактивная карта России (Choropleth):** Географическое распределение кластеров с детализацией по наведению.
    *   **Интерактивный PCA:** 2D-проекция многомерных данных для оценки качества разделения групп.
    *   **Тепловые карты и радары:** Детальный анализ факторных различий между кластерами.
*   **Система верификации:** Встроенный модуль автоматизированного тестирования на базе **pytest**.

## 🛠 Технологический стек
*   **Язык программирования:** Python 3.10+
*   **Анализ данных:** Pandas, Scikit-learn (KMeans, PCA, StandardScaler)
*   **Визуализация:** Plotly (интерактив), Seaborn, Matplotlib
*   **Интерфейс:** Streamlit
*   **База данных:** SQLite3
*   **Инфраструктура:** Docker, Docker Compose

## 📁 Структура проекта
```text
├── app.py                # Точка входа в веб-интерфейс Streamlit
├── services/             # Бизнес-логика (UniversalClusterAnalyzer)
├── models/               # Слой данных и сервисы доступа к БД
├── src/                  # Модули визуализации и предобработки
├── tests/                # Сценарии модульного тестирования (Pytest)
├── utils/                # Вспомогательные утилиты и тестовый раннер
├── data/                 # Локальные хранилища БД и GeoJSON
├── output/               # Автоматически генерируемые отчеты (Excel, PNG, HTML)
└── docker-compose.yml    # Конфигурация контейнеризации
```

## 📦 Быстрый запуск (Docker)

1. Убедитесь, что у вас установлен Docker и Docker Compose.
2. Соберите и запустите проект:
   ```bash
   docker-compose up --build
   ```
3. Откройте приложение в браузере по адресу: `http://localhost:8501`

## 🧪 Тестирование
Система включает модуль автоматизированной проверки математического ядра. Запуск тестов возможен двумя способами:
1. **В интерфейсе:** Вкладка «🛠️ Тестирование» → кнопка «Запустить системные тесты».
2. **В консоли контейнера:**
   ```bash
   docker exec klasteringvkr-app-1 python -m pytest tests/
   ```

## 📊 Результаты анализа
После каждого запуска система генерирует структурированный отчет в папке `output/`, включающий:
*   Таблицы принадлежности регионов к кластерам (`.xlsx`).
*   Матрицы средних значений факторов по группам.
*   Интерактивные HTML-модели (Карта РФ, PCA).
*   Статические графики для вставки в отчетную документацию.
