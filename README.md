# 📊 Информационно-аналитическая система кластеризации регионов РФ по уровню цифровизации малого бизнеса

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/)
[![SQLite](https://img.shields.io/badge/SQLite-3-lightgrey.svg)](https://www.sqlite.org/)
[![scikit-learn](https://img.shields.io/badge/scikit--learn-Machine%20Learning-orange.svg)](https://scikit-learn.org/)
[![Pandas](https://img.shields.io/badge/Pandas-Data%20Analysis-150458.svg)](https://pandas.pydata.org/)

Программно-аналитический комплекс для многоуровневой классификации субъектов и федеральных округов Российской Федерации на основе показателей цифровой зрелости малых предприятий (МСП). 

Проект разработан в рамках выпускной квалификационной работы (НИЯУ МИФИ).

## 🎯 Описание проекта

Традиционные методы оценки цифрового неравенства (рейтинги путем суммирования показателей) обладают высокой долей субъективности и маскируют структурные проблемы регионов. Данная система решает эту проблему, используя методы машинного обучения без учителя для поиска объективных скрытых закономерностей.

Система анализирует **11 ключевых факторов цифровизации** (от широкополосного интернета и маркетплейсов до внедрения ИИ, Big Data и промышленных роботов) по **87 субъектам РФ** (на базе данных Росстата за 2024 год).

### Ключевые особенности системы:
* **Интеллектуальная обработка пропусков:** Реализован механизм *пенализации* отсутствующих статистических данных (замена нулями/минимумами вместо усреднения) для предотвращения искусственного завышения рейтинга отстающих регионов.
* **Математическое обоснование:** Оптимальное количество кластеров (k=3) доказано аппаратно с помощью **Метода локтя (Elbow method)**.
* **Снижение размерности:** Использование метода главных компонент (**PCA**) для сжатия 11-мерного пространства признаков в 2D-плоскость с целью визуальной верификации качества кластеризации.
* **Автономность:** Использование бессерверной СУБД **SQLite** (БД приведена к 3НФ) обеспечивает полную портативность приложения.

## 🛠 Технологический стек

* **Язык программирования:** Python 3.x
* **База данных:** SQLite3
* **Машинное обучение:** `scikit-learn` (K-Means++, PCA, StandardScaler)
* **Анализ данных:** `pandas`, `numpy`
* **Визуализация:** `matplotlib`, `seaborn`

## 📈 Результаты кластеризации

Алгоритм `K-means++` разделил регионы на 3 макро-группы с ярко выраженными экономическими профилями:

1. **Передовые субъекты (37 регионов):** Лидеры по внедрению наукоемких технологий (ИИ, ЦОД, Big Data).
2. **Субъекты с потенциалом развития (18 регионов):** Находятся в "ловушке среднего уровня" — высокая активность на маркетплейсах при критическом отставании во внутренней автоматизации (ERP, CRM, облака).
3. **Субъекты-аутсайдеры (32 региона):** Испытывают системный дефицит базовой инфраструктуры (ШПД, веб-сайты). 

Для каждого кластера в рамках работы сформированы адресные рекомендации по мерам государственной поддержки.
